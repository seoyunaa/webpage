윤산일기 (云山日记) 프로젝트

리포지터리 확장과 분석 파이프라인의 최신 공개 업데이트

이 페이지는 윤산일기 (云山日记) 프로젝트의 현재 공개 업데이트를 정리한 공간입니다. 리포지터리는 이제 단순한 텍스트 정리 단계를 넘어, 원문 보존, 구조화 메타데이터, 사건 중심 모델링, 분석용 산출물을 연결하는 층위형 DH 워크플로로 확장되었습니다.

  • 초점: 원대 강남, 일기 기반 연구 데이터, 사건 중심 해석
  • 현재 공개 범위: 메타데이터 워크플로, 그래프 산출물, semantic/RAG 레이어, publication assets, Neo4j 패키지
  • 상태: 리포지터리와 워크플로 확장에 대한 공개 업데이트 진행 중

개요

프로젝트 범위

이 프로젝트는 윤산일기 (云山日记)와 관련 원대 강남 자료를 중심에 둡니다. 단순히 일기 텍스트를 보존하는 데 그치지 않고, 이를 구조화된 연구 데이터로 전환하여 역사 분석, 공간 분석, 네트워크 분석에 활용할 수 있도록 하는 것이 목적입니다.

공개 프레이밍

이번 업데이트에서는 리포지터리를 하나의 완결된 DH 연구 파이프라인으로 제시합니다. 워크플로는 원문 보존과 주석 작업에서 출발해, 파생 사건, 그래프 생성, semantic exports, publication용 도표, Neo4j 기반 탐색 분석으로 이어집니다.

최근 업데이트

리포지터리 확장

최근 Codex 작업 주기 동안 Yunshan Diary 리포지터리는 단순한 텍스트 추출 작업 공간에서 층위형 DH 파이프라인으로 확장되었습니다. 현재 워크플로는 원문 보존, 기계 추출, 수동 주석, 통합 delivery records, 파생 사건 레이어, knowledge-graph exports, semantic assertions, RAG chunks, JSON-LD/RDF outputs, chapter-oriented analysis assets를 모두 포괄합니다.

실제 연구 활용 측면에서 보면, 개별 일기 항목은 원문에 계속 연결된 상태를 유지하면서도 구조화 메타데이터, 분석용 사건, 그래프 관계, publication용 시각 자료, Neo4j용 네트워크/공간 데이터로 동시에 전환될 수 있게 되었습니다.

  • 파생 사건 리듬 및 domain 분석 산출물
  • knowledge-graph node와 edge exports
  • semantic assertions, relation instances, JSON-LD/RDF exchange files
