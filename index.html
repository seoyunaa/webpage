Digital Humanities

Yunshan Diary (云山日记) Project

Centered on Yunshan Diary (云山日记), this project organizes source materials, research metadata, and digital outputs related to Yuan Jiangnan and the Yunshan Diary.

  • Approach: source curation, metadata organization, and digital publication planning
  • Status: research dataset and project structure in progress
  • Output: Project Notes

Mongol Princess Project

This project examines Mongol princesses and marriage alliances in the Mongol Empire and Yuan China through structured biographical data, kinship relations, and public-facing outputs.

  • Approach: prosopographical research, relational data structuring, and visualization planning
  • Status: dataset design and project framework in progress
  • Output: Project Notes
